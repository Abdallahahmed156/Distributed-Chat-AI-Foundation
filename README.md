# Distributed AI-Ready Chat Infrastructure 

This project is a high-performance **Distributed Systems** foundation designed to support real-time communication between multiple clients and a central server. It serves as the **Scalable Infrastructure** for building an AI-powered Chatbot system.

## Why this is an AI Foundation?
- **Asynchronous Architecture:** Built with multi-threading to handle multiple user queries to an AI model simultaneously.
- **Centralized Brain (Server):** The server is designed to act as the hosting node for AI models (like GPT or custom LLMs) in future iterations.
- **Identity Tracking:** Features unique identity prefixing for each node, essential for maintaining context in AI conversations.

## Core Features
- **Multi-Client Concurrency:** Supports a distributed network of users.
- **Real-Time Monitoring:** A professional GUI for the server to track data flow and system logs.
- **Custom Handshake:** Personalized welcome protocols for every connected node.
- **Clean OOP Design:** Built for modularity, making it easy to integrate AI processing functions.

## Project Structure
1. **server.py**: The backbone/orchestrator of the distributed network.
2. **client.py**: The interface for interacting with the system.
3. **config.py**: Centralized network configuration.
